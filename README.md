# 📊 IPL Batsman Performance Analysis

This project focuses on analyzing the performance of top IPL batsmen using Python-based data analytics techniques. The dataset includes key batting metrics such as Runs, Matches, Batting Average, Strike Rate, Centuries (100s), and Boundaries (4s and 6s).

The analysis was carried out using libraries like **Pandas**, **Matplotlib**, and **Seaborn** to extract meaningful insights and visualize player performance patterns.

## 🔍 Key Analyses Performed

* **Top 10 Run Scorers**
  Identified and visualized the leading run scorers using a bar chart to understand consistency and dominance.

* **Strike Rate vs Batting Average**
  Used a scatter plot to explore the relationship between scoring speed and consistency, helping classify players as aggressive or stable.

* **Correlation Heatmap**
  Analyzed relationships between multiple performance metrics (Runs, Matches, 100s, 6s, etc.) to identify strong and weak correlations.

* **Distribution of Centuries**
  Used histogram and KDE plots to understand how centuries are distributed among players and identify elite performers.

* **Boundary Contribution Analysis**
  Calculated the percentage of runs scored through boundaries (4s and 6s) to identify explosive hitters versus anchor players.

## 📈 Insights

* High run scorers tend to have strong consistency over multiple matches.
* Strike rate and average reveal different playing styles (aggressive vs stable).
* Only a small group of players dominate in terms of centuries.
* Boundary contribution highlights players who rely heavily on power hitting.

## 🛠️ Tools & Technologies

## 🚀 Conclusion

This project demonstrates how data analysis can be used to evaluate sports performance, uncover patterns, and derive actionable insights. It provides a foundational approach to exploratory data analysis (EDA) using real-world cricket data.
